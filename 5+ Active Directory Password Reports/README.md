# Get Active Directory Users' Password Expiration Date Using PowerShell

Retrieving password expiry date and last password change date for all Active Directory users using native tools can be time-consuming and inefficient. This script simplifies the process by retrieving password last set and password expiry details for all users in a single run, and also allows you to filter results for password-expired accounts, users with soon-to-expire passwords, and more.
