## Identify Dormant Users in Active Directory Using PowerShell  
Identifying inactive users based on their true last logon time using native Active Directory tools can be time-consuming and inefficient, especially in environments with multiple domain controllers.

This script simplifies the process by querying all domain controllers to retrieve the true last logon time and accurately identify inactive users. It also allows you to generate targeted reports such as inactive enabled users, inactive disabled users, never logged-in users, and more.
